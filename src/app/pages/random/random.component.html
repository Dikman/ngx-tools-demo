Random helper

Random.number(min, max) generates a random integer from a given range (a result can include at both the minimum and the maximum of the range).

Arguments

Name Type Description
min number Low integer value of the range
max number High integer value of the range

Example

Life Demo

Open the Developer Console panel and open the Console tab. You can see a result there when you are moving the mouse pointer over the event listener area below.
